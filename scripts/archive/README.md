Archived scripts
================

This folder contains older scripts kept for reference. Prefer these up-to-date commands:

- Start (foreground): `./start_app.sh`
- Start (background): `./start_orchestrator_background.sh`
- Stop (background): `./stop_orchestrator_background.sh`
- Status: `./status.sh`

The archived `start_full.sh` mixed hard-coded paths/ports and can drift from current config. Use the commands above instead.
